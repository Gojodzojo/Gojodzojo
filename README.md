
# Hi, I'm Mateusz Goik

I have been programming almost daily since 2018. Most of time I make web applications (both on frontend and backend), but I also like working with low level languages and embedded systems. I love learning new things and I'm always trying to improve my skills.

Me and my friends from [KNEI](https://github.com/knei-knurow) took part in the [cansat competition](https://esero.kopernik.org.pl/konkurs-cansat/) (organised by European Space Agency) and we received a distinction.

Now we are working on a mars rover for  the [European Rover Challenge](https://roverchallenge.eu/).
